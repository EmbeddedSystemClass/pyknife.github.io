# 【开源四足机器人】 HCRT-DOG

![Jump](./img/Jump.gif)


**本项目是一个并联式的四足机器人 每条腿使用了两个电机驱动共四条腿，八个电机期望实现四种步态，从慢到快依次为，行进步态（三足支撑）、小跑步态（两拍步态）、溜蹄步态、疾驰步态。**

+ 如果说全程使用跳跃的步态，对于电机的损伤也是非常的大的，在调试的时候要非常注意电机的电流，做电流检测。
